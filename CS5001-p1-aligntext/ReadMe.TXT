This is a programme to align text.

usage: 

java AlignText file_name line_length
or
java AlignText file_name line_length [align_mode]

Three arguments can be specified. The first one is used as the path of the original text. The second one is used to specify the length and the third one is used to assign allign mode.

L(default): left-align mode
R: rignt-align mode
C: centre-align mode
B: surround text with bubble 
other: left-align mode

If the third argument is missing, it runs default left-align mode.
